# Erlang 变量重复赋值检测工具 - 使用说明

## 快速开始

```bash
# 检测单个文件
python scripts/check_variable_reassignment_v2.py src/boot/test_repeated_assignment.erl

# 检测整个目录
python scripts/check_variable_reassignment_v2.py src/

# 检测多个路径
python scripts/check_variable_reassignment_v2.py src/boot/ src/core/

# 调试模式（查看详细信息）
python scripts/check_variable_reassignment_v2.py -v src/problem_file.erl

# 输出到文件
python scripts/check_variable_reassignment_v2.py src/ > report.txt 2>&1
```

## 集成到 Makefile

在 `makefile` 中添加：

```makefile
.PHONY: check-vars
check-vars:
	@python scripts/check_variable_reassignment_v2.py src/
```

使用：
```bash
make check-vars
```
